# EXCEPTIONS.md
**Registered exceptions to `AZHUB_V2_CONSTITUTION.md` (per CHG-R-002)**

Any rule violation that the owner has explicitly approved MUST be recorded here.
An undocumented exception is a violation, not an exception.

## Template

```
### <ID>: <Short title>
- **Date approved:** YYYY-MM-DD
- **Rule(s) waived:** e.g. DB-R-040, UX-R-008
- **Scope:** <which files / routes / environments are affected>
- **Reason:** <why the strict rule is being waived>
- **Expiry:** <date or "permanent"; prefer a date>
- **Rollback plan:** <how to restore strict compliance later>
- **Approver:** <owner signature / GitHub handle>
```
